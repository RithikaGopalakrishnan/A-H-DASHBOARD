A&H Dashboard

A&H Dashboard is a modern, user-friendly web application built with Angular and Firebase to manage video content efficiently. It allows you to upload, view, edit, and organize videos with a clean interface and real-time updates.

Project Overview

This dashboard is designed for smooth video content management. With real-time Firebase synchronization, all changes are instantly reflected, making it easy to keep content up-to-date. It works seamlessly across desktop, tablet, and mobile devices.

Key Features

Upload & Manage Videos: Add new videos, edit metadata, or delete content easily.

Responsive Dashboard: Clean sidebar and header for smooth navigation.

Real-time Updates: All changes sync instantly with Firebase.

Automatic Date Formatting: Upload dates are neatly displayed for clarity.

Technology Stack

Frontend: Angular 19.2.0 with TypeScript

Backend: Firebase Firestore

UI Framework: Angular Material

Styling: SCSS for responsive design

Build Tool: Angular CLI

Getting Started

Clone the repository

git clone https://github.com/RithikaGopalakrishnan/A-H-DASHBOARD.git
cd A-H-DASHBOARD


Install dependencies

npm install


Configure Firebase

Set up a Firebase project with Firestore.

Add your Firebase config to src/environments/environment.ts.

Run the app

ng serve


Open http://localhost:4200/ in your browser.
